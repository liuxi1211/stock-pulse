# 个人因子选股策略避坑清单
（面向中低频、日线级、个人开发者，核心目标：让策略从「回测好看」变成「实盘能用」，避开90%的个人量化常见陷阱）

---

## 一、回测阶段：90%的“好策略”，死在这一步
回测是量化的第一道关，也是最容易自欺欺人的环节。绝大多数人最终实盘翻车，根源都在回测阶段就埋好了。

### 1. 头号死敌：过拟合（数据挖掘偏差）
- **坑的本质**：通过反复调整因子、参数，让策略完美贴合过去的历史行情。看起来收益爆表，本质是用未来的参数“预测”过去，相当于开着后视镜开车，一到实盘就彻底失效。
- **典型过拟合特征（对照自查）**：
    1. 参数极度敏感：核心阈值上下浮动10%，年化收益就从25%掉到5%；
    2. 样本内封神，样本外崩掉：比如2010-2020年回测年化30%，2021-2025年纯测试期连指数都跑不赢；
    3. 交易信号极其稀少：全市场几年才出几十次信号，刚好全部命中历史低点；
    4. 因子数量≥8个，且多个因子逻辑高度重叠。
- **规避方法（可直接落地）**：
    1. **严格样本外验证**：历史数据按时间切分，前80%用来调参数，后20%做纯测试集，测试集绝对不能碰任何参数调整；
    2. **滚动回测（Walk Forward）**：最接近实盘的验证方式。比如每2年校准一次参数，用接下来1年做验证，逐年滚动推进；
    3. **参数敏感性测试**：把核心阈值上下浮动20%，若收益波动超过30%，说明策略稳定性极差，不能用；
    4. 因子数量控制在3-5个，每个因子分属不同逻辑大类。

### 2. 收益幻觉：忽略交易成本与现实约束
很多人回测年化25%，实盘只剩8%，差的部分全是没算进去的成本。
- **必须足额计入的成本（A股标准）**：
  | 成本项 | 标准取值 | 说明 |
  |---|---|---|
  | 手续费 | 双边约0.2% | 印花税0.1%（卖出）+ 佣金万1左右（买卖双向） |
  | 滑点 | 大盘股0.1% / 中证500成分股0.2% / 小票0.3%-0.5% | 实际成交价格和挂单价格的差值，市值越小滑点越大 |
  | 冲击成本 | 单日成交额<5000万的个股额外加0.3%以上 | 大资金买入会推高股价，个人小资金可适当降低 |
- **必须考虑的现实约束**：
    - 涨跌停：不能默认涨跌停都能成交，实盘涨停买不到、跌停卖不出是常态；
    - 停牌：遇到长期停牌股，资金被锁定，回测要计入流动性损失；
- **硬性经验法则**：回测出来的年化收益，先打7-8折，才是你实盘大概率能拿到的真实收益。

### 3. 幸存者偏差：你选的都是“活下来的赢家”
- **坑的本质**：用当前的指数成分股（比如当下的沪深300）回测过去10年，等于默认10年前你就知道哪些公司能活到现在、能进入指数。这是开上帝视角选股，回测收益必然虚高。
- **规避方法**：
    1. 股票池用全市场个股，包含退市、ST、摘牌标的；
    2. 如果只用指数成分股，必须用对应日期的**历史成分股**，绝对不能用当前成分股回测历史。

## 二、因子构建：几十个因子，不如3个逻辑过硬的因子
你提到“几十个常用不常用因子搭配”，这里是第二个重灾区。因子不是越多越好，多了反而会互相干扰、放大过拟合。

### 1. 因子数量陷阱：多重比较偏差
- **底层逻辑**：如果你随机试100种因子组合，哪怕全是毫无逻辑的随机因子，也至少有5种能跑出“年化20%、胜率60%”的结果。这是统计概率，不是策略有效。你试的组合越多，得到假阳性结果的概率越高。
- **精简原则**：
    1. 按逻辑大类划分：**价值、动量、质量、波动率**，每大类最多选1个核心因子；
    2. 同逻辑因子不要重复叠加：比如PE、PB、PS都属于价值类，选一个就行；MACD、均线多头、阶段涨幅都属于动量类，选一个就行；
    3. 黄金标准：3个核心因子为最佳，最多不超过5个。

### 2. 逻辑优先：拒绝“先有数据，后编故事”
这是区分「科学量化」和「数据挖矿」的核心标准。
- **错误顺序**：跑一遍全市场几百个因子，找出过去5年收益最高的10个，拼在一起，然后给它们编一个“价值成长结合”的理由；
- **正确顺序**：先从商业规律/市场逻辑出发提出假设 → 再用回测验证假设 → 证伪就放弃，证实就保留。
  比如：
    - 假设：低估值的股票长期会向合理估值回归 → 验证低PE因子；
    - 假设：上涨趋势的个股会延续趋势 → 验证动量/均线因子。
- **A股长期被实证有效的四类核心因子**：
    1. 价值类：低PE、低PB、高股息（赚估值修复的钱）
    2. 动量类：阶段涨幅、均线多头、MACD趋势（赚趋势延续的钱）
    3. 质量类：高ROE、净利润增速稳定、现金流健康（赚企业成长的钱）
    4. 波动率类：低波动、低换手（赚避险溢价的钱）

### 3. 中性化处理：对冲板块与情绪干扰
这直接解决你担心的「A股情绪和板块影响大」的问题。
- **为什么必须做**：如果全市场选低PE，最后选出来的大概率全是银行、地产、煤炭。你本质不是在选股，是在押注价值板块。一旦板块风格切换，策略会直接崩盘，看起来就像“因子失效了”。
- **个人可落地的2种中性化方法（不用复杂数学）**：
    1. **行业中性**：在每个申万一级行业内部按因子排名打分，每个行业选排名靠前的N只，最终各行业持仓比例和宽基指数基准接近，彻底规避单一行业暴雷；
    2. **市值中性**：只在固定市值区间选股（比如100亿-500亿），或按市值分层选股，避免持仓全是小票或全是大票。
- **进阶玩法**：如果想赚板块轮动的钱，单独加一个**板块动量因子**（比如选近1个月涨幅前3的行业），把情绪轮动也变成规则的一部分，而不是靠主观判断。

### 4. 条件越复杂，死得越快
- 反例：MACD金叉 + KDJ金叉 + RSI<30 + PE<20 + PB<1 + 成交量放大2倍。限制条件越多，历史上符合条件的样本越少，越容易是偶然结果，未来越难重复；
- 正例：PE处于行业前30% + 近1个月涨幅处于行业前40%。两个条件，逻辑清晰，样本充足，鲁棒性极强。

## 三、实盘落地：回测年化30%，实盘可能亏10%
回测再完美，实盘才是最终检验。这一步的核心不是赚多少钱，是**验证策略的可复制性**。

### 1. 验证期铁则：100%对齐回测，禁止主观干预
- **验证期配置**：
    - 资金比例：总资金的10%-20%，绝对不能一开始就满仓；
    - 验证时长：至少6个月，最好穿越一次市场风格切换；
- **核心禁令**：验证期内，100%严格执行策略信号，禁止主观筛选个股、提前止盈、临时补仓、手动止损。
  一旦你主观干预，你就永远分不清赚钱是靠策略还是靠感觉，整个验证过程完全失效。

### 2. 实盘偏差跟踪：精准定位问题
每个月固定复盘3个核心偏差，快速定位问题：
1. **收益偏差**：实盘收益 vs 同期回测收益，差值持续超过±5%就要排查原因；
2. **成交偏差**：策略发出信号的股票，实际成交比例有多少？有没有因为涨跌停、流动性不足买不到的情况；
3. **持仓偏差**：实际持仓和策略推荐持仓的重合度，有没有低于90%。
- 常见偏差原因：滑点设低了、个股流动性差、交易成本算少了、主观换股。

### 3. 仓位风控：别让策略死在一次黑天鹅里
- 持股数量：单期持仓10-20只为宜，太少分散不足，太多个人管不过来；
- 单票上限：单只股票仓位不超过总资金的10%；
- 行业上限：单个行业仓位不超过总资金的20%；
- 止损规则：必须写进策略代码里，比如单只浮亏20%强制止损，或对应宽基指数回撤超过15%时整体降仓到50%，不要靠心态硬扛。

## 四、长期迭代：别让“优化”变成“自杀”
很多人不是死在策略差，是死在不停改策略。

### 1. 正确区分「风格回撤」和「策略失效」
A股风格轮动极强，任何策略都会有半年到一年的跑输期，这是正常现象，不是策略没用。
- **正常的风格回撤**：
    1. 策略的核心逻辑没有被破坏（比如低PE的估值修复逻辑依然成立）；
    2. 只是当前市场风格不匹配（比如全市场炒题材，价值股跑输）；
    3. 回撤幅度没有超过历史最大回撤；
    4. 持续时间不超过12个月。
       → 应对：坚持执行，风格总会轮动回来。
- **真正的策略失效**：
    1. 核心逻辑被彻底破坏（比如打新策略在破发常态化后失效）；
    2. 回撤大幅超过历史最大回撤，且持续创新低；
    3. 样本外连续18个月跑不赢基准，且没有回暖迹象。
       → 应对：停用该策略，重新开发。

### 2. 迭代红线：禁止频繁调参
- 调参频率上限：**最多1年调一次**，绝对不能月月调、周周调；
- 调参数据要求：必须用至少5年的历史数据校准，不能只用最近1年的行情；
- 正确的迭代姿势：不是反复微调阈值，而是补充新的逻辑维度（比如原来只有价值+动量，现在加入质量因子过滤暴雷股）。

### 3. 放弃圣杯思维
- 没有任何策略能年年跑赢大盘。能做到10年里7年跑赢，长期年化比宽基指数高出5%-10%，就是顶级的个人策略；
- 不用羡慕机构的大模型、高频策略。你们赚的不是同一笔钱：他们赚毫秒级价差的钱，你赚企业基本面和风格贝塔的钱，赛道完全不同，不存在降维打击。

## 五、绝对红线：碰了必亏的5件事
1. **绝对不能用未来函数**：比如用当日收盘价生成当日买入信号、用未来的财报数据做过去的选股。这是回测作弊，实盘100%亏；
2. **绝对不能为了回测好看反复优化参数**：多调一次参数，过拟合程度就加一分；
3. **绝对不能策略没验证就满仓加杠杆**：量化不是保本神器，杠杆会放大回撤，一次极端行情就爆仓；
4. **绝对不能实盘主观干预策略**：一边写代码一边凭感觉买卖，等于白做量化；
5. **绝对不要碰日内高频、T0策略**：个人没有通道、算力和数据优势，只会被机构收割。

## 附：快速自检表（10条全打勾才算合格）
□ 1. 回测做了严格的样本外验证，测试集未调整过任何参数
□ 2. 回测足额计入了手续费、滑点和冲击成本
□ 3. 股票池包含退市/ST股，未用当前成分股造成幸存者偏差
□ 4. 核心因子不超过5个，每个因子都有清晰的经济逻辑
□ 5. 因子分属不同大类，没有同类型因子重复叠加
□ 6. 做了行业中性/市值中性处理，无单一行业过度暴露
□ 7. 策略逻辑简单清晰，没有叠加7个以上的复杂条件
□ 8. 实盘验证期用小资金，100%严格执行信号
□ 9. 有明确的仓位分散和止损规则，且已写入策略
□ 10. 接受策略会阶段性跑输，不追求年年翻倍
