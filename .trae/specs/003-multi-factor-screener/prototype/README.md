# 多因子选股中心 · 前端页面原型

本目录存放「多因子选股中心」的静态 HTML 原型，用于需求评审与视觉方向确认。

## 文件说明

| 文件 | 说明 |
|---|---|
| `index.html` | 原型主页面，包含方案库、规则树编辑器、过滤、排序与结果面板 |
| `styles.css` | 设计样式：纸本草稿纸质感 + 印章红强调色 + 证券报排版 |
| `app.js` | 原型交互逻辑与模拟数据 |
| `design.md` | 设计令牌、布局概念、自我评审与风险点 |

## 如何查看

直接用浏览器打开 `index.html` 即可，无需构建或后端服务。

## 原型覆盖的功能

1. **选股模式切换**：快照选股 / 区间选股。
2. **方案库**：左侧方案卡片列表，支持新建、选择方案。
3. **条件规则树**：AND/OR 分组、比较叶子节点（因子 vs 数值/因子）、添加/删除节点。
4. **静态过滤**：ST、停牌、涨跌停、行业白/黑名单、上市天数。
5. **排序打分**：单因子排序、多因子 z-score 加权综合打分、top_n 截断。
6. **结果面板**：
   - 快照：命中名单表格 + 排名 + 得分 + 因子值 + 过滤明细。
   - 区间：每只股票的命中分布、首次命中日、连续命中天数。
   - 追踪：锁定结果后展示 5/10/20 日等权组合收益。
7. **印章运行按钮**：点击「运行选股」触发印章动画，象征决策落库。

## 设计要点

- 采用「证券报纸 + 交易大厅公告板」的隐喻，用纸张底色与精细网格传递研究的严谨感。
- 字体组合：`Noto Serif SC`（标题/方案名）+ `Noto Sans SC`（正文）+ `JetBrains Mono`（代码/数值）。
- A 股约定：红色表示上涨/主操作，绿色表示下跌/通过态。
- 已做移动端适配、`prefers-reduced-motion` 与键盘焦点样式。

## 注意

本原型为**纯前端演示**，所有选股结果均为本地模拟数据，不连接 stock-engine / stock-watcher 服务。
