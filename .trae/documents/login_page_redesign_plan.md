# StockPulse 登录页面改造计划

## 一、项目背景与目标

将原有的简单登录页面改造为符合新设计系统的现代化登录页，采用深色科技风 + 玻璃拟态风格，提升品牌形象和用户体验。

## 二、现状分析

### 当前 login.html 结构
- 单卡片居中布局
- 仅引入 custom.css，未使用新的主题系统
- 基础的 Bootstrap 表单样式
- Thymeleaf 变量：`${error}`、`${message}`、`${username}`
- 登录表单：POST 到 `/login`，字段 `username`、`password`

### 设计系统资源
- **theme.css**: 深色/浅色双主题变量系统，包含渐变、玻璃拟态、光晕效果
- **components.css**: 按钮、表单、卡片等组件样式
- **common.html**: 包含主题切换 JS 逻辑参考

## 三、改造方案

### 3.1 整体布局
- 左右分栏布局（左侧品牌区 + 右侧登录卡片）
- 全屏高度，响应式适配（移动端单列）
- 背景：深色渐变 + 多层光晕效果

### 3.2 左侧品牌展示区
- 大 Logo（StockPulse 图标 + 文字）
- Slogan 标语
- 装饰性元素（渐变光斑、网格线、浮动卡片等科技感元素）
- 特性亮点列表（可选）

### 3.3 右侧登录卡片
- glass 玻璃拟态效果（backdrop-filter + 半透明背景）
- 卡片标题 + 欢迎语
- 错误提示 / 成功提示（保留 Thymeleaf 变量）
- 用户名输入框（带图标）
- 密码输入框（带图标 + 显示/隐藏切换）
- 渐变色主按钮登录
- 底部辅助链接（忘记密码等，可选）

### 3.4 主题切换
- 右上角主题切换按钮（深色/浅色）
- 复用 theme.js 的主题切换逻辑
- 主题切换时背景和卡片样式自动适配

### 3.5 样式规范
- 所有颜色使用 theme.css 中的 CSS 变量
- 按钮、表单使用 components.css 中的组件类
- 动画效果使用 theme.css 中定义的 transition 和 animation

## 四、文件修改清单

| 文件 | 操作 | 说明 |
|------|------|------|
| `templates/pages/login.html` | 重写 | 完整重构登录页面 |

## 五、保留的功能点

1. ✅ Thymeleaf 模板变量：`${error}`、`${message}`、`${username}`
2. ✅ 登录表单 POST 到 `/login`
3. ✅ 用户名字段 `username` + 密码字段 `password`
4. ✅ 表单验证（required、autofocus）

## 六、技术实现要点

1. **引入新样式**：替换 custom.css 为 theme.css + components.css
2. **主题初始化**：在 head 中引入 theme.js
3. **玻璃拟态**：使用 `.glass` 类 + 自定义增强样式
4. **背景光晕**：使用 CSS radial-gradient 多层叠加
5. **装饰元素**：使用 CSS 伪元素和绝对定位
6. **响应式**：移动端左侧品牌区隐藏或堆叠在上部

## 七、预期效果

- 视觉上：深色科技风、玻璃拟态、渐变光晕，符合现代金融科技产品调性
- 功能上：完整保留原有登录逻辑和 Thymeleaf 变量
- 体验上：流畅的过渡动画、主题切换、输入反馈
