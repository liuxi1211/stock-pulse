## 3. 手动验证清单（必填项必须验证）

### 3.1 必走流程 1 — 新用户首次使用

```
1. 启动系统，等待数据初始化完成
2. 使用 admin / admin123 登录
3. 完成 TOTP 两步验证设置（扫描 Google Authenticator）
4. 在搜索框输入"平安银行"或"000001"，点击搜索
5. 从搜索结果点击进入个股详情页
6. 查看 K线图，切换日/周/月线周期
7. 点击"加入自选"
8. 打开"自选股"页面，确认股票已在列表中
9. 从自选股列表点击"移除"，确认股票已被删除
10. 点击右上角用户名 → 退出登录
```

### 3.2 必走流程 2 — 管理员操作

```
1. 使用 admin 账号登录
2. 进入"用户管理"页面
3. 创建一个新用户 testuser（普通用户角色）
4. 退出 admin，用 testuser 登录
5. 尝试访问用户管理页面 → 确认被拒绝（403）
6. 添加几只股票到自选股
7. 退出 testuser
8. 用 admin 登录，删除 testuser 账号
9. 尝试用 testuser 登录 → 确认无法登录
```

### 3.3 必走流程 3 — 数据更新与校验

```
1. 观察 16:00 定时任务日志，确认数据更新正常
2. 观察 22:00 数据校验日志，确认无异常报告
3. 手动触发一次数据 API 调用统计页面，确认接口正常工作
```
