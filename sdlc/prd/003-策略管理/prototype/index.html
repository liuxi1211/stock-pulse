原型使用说明

📋 前端开发参考

  • 使用 Bootstrap 5.3.3 样式体系，与选股模块保持一致
  • 侧边栏导航 + 顶部工具栏的经典布局
  • 编辑器采用「单 JSON 对象为单一数据源」原则
  • 表达式编辑器是递归组件：因子引用/静态值/算术表达式
  • 规则树深度限制：嵌套组 ≤ 3 层，算术表达式 ≤ 3 层
  • 操作符选择 cross_up/cross_down 时锁定左右为因子引用

🔧 后端开发参考

  • 主表：quant_strategy，版本表：quant_strategy_version
  • 状态流转：DRAFT → ACTIVE（通过回测验证）→ ARCHIVED
  • 保存为策略时自动写入 version 快照表
  • Python 端 RuleParser 产出：FactorSpec + ConditionEvaluator + ParamPath
  • 因子注册表由 Java 缓存，前端页面加载时获取一次
  • cross_up/cross_down 的实现需访问上一根 bar 的值
